# Airline Booking System

A console-based airline booking and flight management system built with Java.

## Overview
This project simulates an airline reservation system where employees can manage flights and passengers can book or cancel reservations.

## Features
- Add and remove flights
- Display available flights
- Book flight tickets
- Cancel bookings
- View booking details
- Track shipments
- Employee and passenger menus

## Tech Stack
- Java
- Object-Oriented Programming (OOP)

## OOP Concepts Used
- Abstraction
- Inheritance
- Polymorphism
- Encapsulation
- Interfaces

## Project Structure
- `Flight` (Abstract Class)
- `Economy`, `Business`, `FirstClass`
- `Passenger`
- `Employee`
- `Booking`
- `Shipment`
- `Trackable` (Interface)

## How to Run
1. Open the project in NetBeans or any Java IDE
2. Compile and run `Main.java`
